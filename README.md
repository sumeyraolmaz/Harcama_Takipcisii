# Expense Tracker App 📊

This project is a comprehensive expense tracking application built with React. It allows users to manage their income and expenses effectively by providing features for creating, reading, updating, and deleting expense records. The application includes filtering, sorting, and dark mode functionalities to enhance user experience. It also displays expense statistics to provide insights into spending habits.

## 🚀 Key Features

- **Data Fetching:** Retrieves expense data from a backend API. 📡
- **Filtering:** Filters expenses by category, search term, and date range. 🔍
- **Sorting:** Sorts expenses by date (newest or oldest). ⬆️⬇️
- **CRUD Operations:** Creates, reads, updates, and deletes expenses. ✍️
- **Form Management:** Handles form input and submission for creating and editing expenses. 📝
- **UI Rendering:** Displays the list of expenses, filters, form, and other UI elements. 🎨
- **Dark Mode:** Toggles between light and dark themes. 🌙
- **Expense Statistics:** Displays insightful statistics about expenses. 📈

## 📦 Getting Started

### Prerequisites

- Node.js and npm installed on your machine. 💻
- A backend API running (e.g., on `http://localhost:5000/api/Expenses`). 🌐

### Installation

1.  Clone the repository:

    ```bash
    git clone <repository-url>
    ```

2.  Navigate to the project directory:

    ```bash
    cd <project-directory>
    ```

3.  Install the dependencies:

    ```bash
    npm install
    ```

### Running Locally

1.  Start the development server:

    ```bash
    npm start
    ```

2.  Open your browser and navigate to `http://localhost:3000`. 🌐
